**Day 141 — Execution Log | Date : 11/05/2026**



**Session Type:** Core Session



**Topic:** Thread Pools



**Concept:**

A thread pool is a reusable group of worker threads used to execute tasks efficiently without repeatedly creating new threads.



**Key Learnings:**

* Thread creation has overhead
* Thread pools improve efficiency through reuse
* Controlled concurrency reduces excessive context switching
* Pool sizing affects system performance and scalability



**Core Insight:**

Efficient systems reuse expensive resources instead of constantly recreating them.



**Outcome:**

* Learned practical concurrency optimization
* Connected thread management with scalability and performance
* Improved understanding of resource reuse strategies
