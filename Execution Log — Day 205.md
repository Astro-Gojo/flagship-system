**Day 205 — Execution Log** **| Date : 14/07/2026**



**Session Type:** Micro Session



**Topic:** Write-Back Cache



**Concepts Learned:**

* Write-Back Cache
* Deferred Writes



**Key Insights:**

* Cache updates happen immediately.
* Database writes are delayed.
* Performance improves, but durability risks increase.



**Systems Engineering Principle:**

Delaying work can improve performance, but only if the associated risks are understood and managed.



**Progression:**

Write-Through → Write-Back
