**Day 122 — Execution Log | Date : 22/04/2026**



**Session Type:** Core Session



**Topic:** CPU Scheduling



**Concept:**

CPU scheduling determines which thread runs, when, and for how long.



**Scheduling Algorithms:**

* FCFS (First Come First Serve)
* SJF (Shortest Job First)
* Round Robin (time slicing)
* Priority Scheduling



**Key Learnings:**

* Different algorithms optimize for different goals
* FCFS can delay short tasks
* SJF reduces average waiting time but may starve long tasks
* Round Robin ensures fairness
* Priority scheduling can cause starvation



**Core Insight:**

Scheduling decisions define system performance, fairness, and responsiveness.



**Outcome:**

* Understood multiple scheduling strategies
* Learned trade-offs between fairness and performance
* Corrected misconception about priority and fairness
