**Session Type:** Core Session



**Topic:** Scalability



**Concept:**

Scalability is the ability of a system to handle increasing load effectively.



**Key Learnings:**

* Vertical scaling increases power of a single machine
* Horizontal scaling adds more machines
* Scaling must consider bottlenecks and system design
* Poor scaling leads to inefficiency



**Core Insight:**

Effective scalability requires understanding system limits and optimizing before expanding.



**Outcome:**

* Understood types of scaling
* Connected scalability with system performance concepts
* Strengthened real-world system design thinking
