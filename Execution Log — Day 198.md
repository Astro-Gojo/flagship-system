**Concepts Learned:**

* Connection Pooling
* Connection Reuse
* Database Efficiency



**Key Insights:**

* Opening database connections is expensive.
* Reusing existing connections improves performance.
* Connection pools follow the same reuse principle as thread pools.



**Systems Engineering Principle:**

Expensive resources should be reused whenever possible instead of repeatedly created and destroyed.
